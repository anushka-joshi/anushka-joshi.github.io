# Anushka Joshi — Research Portfolio

A personal academic portfolio website for **Anushka Joshi**, Prime Minister Research Fellow and PhD Scholar at IIT Roorkee.

🌐 **Live site:** `https://<your-github-username>.github.io/portfolio`

---

## 📁 Project Structure

```
portfolio/
├── index.html        # Main HTML file
├── css/
│   └── style.css     # All styles
├── js/
│   └── main.js       # Scroll + navigation logic
└── README.md
```

---

## ✨ Features

- Fixed dark sidebar with profile photo, navigation, and social links
- Hero section with forest background and name overlay
- Research interest cards (Deep Learning, ML, Earthquakes, Computer Vision, Real-World Applications)
- Brief Bio with CV download button
- Academic Odyssey timeline
- Selected Publications list
- Teaching experience
- Contact section
- Smooth scroll & active nav highlighting

---

## 🛠️ Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JavaScript** — no frameworks, no build step
- [Google Fonts — Poppins](https://fonts.google.com/specimen/Poppins)
- [Font Awesome 6](https://fontawesome.com/) for icons

---

## 🚀 Running Locally

Just open `index.html` in any browser — no install or build required.

```bash
open index.html
```

Or use a live-reload dev server:

```bash
npx live-server
```

---

## 🌍 Deploying to GitHub Pages

1. Push this folder to a GitHub repository (e.g. `portfolio`)
2. Go to **Settings → Pages**
3. Under **Branch**, select `main` and folder `/ (root)`
4. Click **Save** — your site will be live at `https://<username>.github.io/portfolio`
